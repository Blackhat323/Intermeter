# \# Intermeter

# 

# A lightweight real-time network speed monitor for Windows, built with PyQt6.

# 

# !\[Python](https://img.shields.io/badge/Python-3.10+-blue)

# !\[Platform](https://img.shields.io/badge/Platform-Windows-lightgrey)

# 

# \## Features

# \- Live download / upload speed (auto-scales Kbps → Mbps → Gbps)

# \- Always-on-top frameless widget mode

# \- Speed history graph

# \- DNS cache flush utility

# \- Connection stabilizer (prevents NIC idle drops)

#
